\section{Contributions}

Consistent with the objectives just described, the specific contributions of the thesis can be summarized as follows:
\begin{itemize}
\item documentation and organization of an updated experimental toolchain, with attention to software dependencies, project compilation, and execution reproducibility;
\item porting and adaptation, in the new environment, of the experimental logic inspired by \cite{segata2023multi-technology}, including reference scenarios, PDR monitoring, and the \emph{SafeSwitch} mechanism;
\item migration from LTE to 5G, in order to support a more current cellular model;
\item definition of an experimental plan and an analysis pipeline suitable for evaluating the correctness of the integration, communication robustness, and the effects on platoon dynamics.
\end{itemize}

Taken together, these contributions amount to a toolchain modernization and experimental validation effort. The thesis therefore assesses how well the mechanism studied in the reference paper can be carried over to a more modern software stack, while preserving its logic and making explicit the technical choices required to reproduce it.
