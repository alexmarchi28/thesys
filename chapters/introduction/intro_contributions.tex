\section{Contributions}

Consistent with the objectives just described, the specific contributions of the thesis can be summarized as follows:
\begin{itemize}
\item documentation and organization of an updated experimental toolchain, with attention to software dependencies, project compilation, and execution reproducibility;
\item import and adaptation, in the new environment, of the experimental logic inspired by \cite{segata2023multi-technology}, including reference scenarios, PDR monitoring, and the \emph{SafeSwitch} mechanism;
\item migration from LTE to 5G, in order to support a more up-to-date cellular model;
\item definition of an experimental plan and an analysis pipeline suitable for evaluating the correctness of the integration, communication robustness, and the effects on platoon dynamics.
\end{itemize}

Taken together, these contributions define a work of infrastructural update and experimental validation. The thesis therefore aims to assess how transferable the mechanism studied in the reference paper is to a more modern software stack, while preserving its logic and making explicit the technical choices required for its reproduction.
