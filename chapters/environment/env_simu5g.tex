\section{Simu5G: 5G Cellular Network Modeling}
\label{env:simu5g}

Simu5G is a library for OMNeT++ designed for the end-to-end evaluation of 5G networks, developed as an evolution of the line of work already started with SimuLTE for LTE and LTE-A networks \cite{virdis2014simulte,nardini2020simu5g}. Simu5G is therefore the most suitable choice for replacing the LTE model used in the reference work with a 5G cellular model.

The choice of Simu5G is tied to its native integration with OMNeT++, which makes it possible to insert it into the same simulation infrastructure already used for the other components. In this way, the parameters of the cellular interface can be configured directly through the \texttt{ini} files, keeping the definition of the experimental scenarios coherent with the rest of the environment.
