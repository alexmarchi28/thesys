\chapter{Preparation of the Experimental Environment}
\label{ch:setup}

This appendix documents how the simulation environment was set up in practice. The goal is to make explicit the procedure through which a toolchain compatible with the multi-technology study was assembled, while remaining consistent with the official instructions of Plexe and its related subprojects.

The description is intentionally technical: it includes workspace organization, compilation sequence, the structure of the simulation files, namely NED, INI, and SUMO configuration files, checks for correct operation, and automation of the run campaigns. Where necessary, deviations from standard procedures are noted, especially for SUMO and OMNeT++.
